# **App Name**: Yaneth Control Bancario

## Core Features:

- Efectivo en Caja: Control del efectivo disponible en la caja de cada agente.
- Fondos en Cuenta Bancaria: Registro y control de los fondos disponibles en las cuentas bancarias asociadas.
- Transacciones en Efectivo: Registro de transacciones realizadas en efectivo: ingresos y egresos.
- Retiros y Depósitos: Registro detallado de retiros y depósitos bancarios realizados por los agentes.
- Transferencias: Registro y seguimiento de transferencias bancarias realizadas entre cuentas.
- Pago a Proveedores: Gestión de pagos a proveedores, registrando fecha, monto y concepto.
- Reportes por Localidad: Generación de reportes detallados por cada localidad, mostrando el flujo de efectivo y las transacciones.

## Style Guidelines:

- Primary color: Light Blue (#B2EBF2), creating a sense of trustworthiness.
- Background color: Very light blue (#E0F7FA) to create a clean user interface.
- Accent color: Darker Blue (#29B6F6) for calls to action and important elements.
- Body and headline font: 'PT Sans', a modern and readable sans-serif.
- Use clear, consistent icons to represent different transaction types (retiros, depósitos, etc.).
- Design a clear and organized layout for easy navigation and data entry. Use distinct sections for each module: efectivo, transferencias, reportes.
- Implement subtle animations to provide feedback on user interactions (e.g., successful transaction, loading data).
